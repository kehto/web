    **C. Integrate migrateAclState() in acl-store.ts (per SH-I04 — SHELL-MIGRATION.md section 1.9):**

    The current `aclStore` uses its own internal data format (Map of InternalAclEntry with Set capabilities). The `migrateAclState()` from @kehto/acl operates on the `AclState` type (a `Record<string, AclEntry>` with bitfield caps). These are DIFFERENT formats.

    The migration must happen at the **serialized** layer — after reading raw JSON from localStorage but before parsing into the internal format. Approach:

    1. Import `migrateAclState` from `@kehto/acl`:
       ```typescript
       import { migrateAclState } from '@kehto/acl';
       ```

    2. In the `load()` function, after `JSON.parse(raw)` produces the entries array, check if any key has the old 3-segment format (`pubkey:dTag:hash`). If so, build an `AclState` object from the entries, call `migrateAclState()`, then reconstruct the entries array from the migrated state and re-persist immediately.

    Specifically, update `load()` to:
    ```typescript
    load(): void {
      try {
        const raw = localStorage.getItem(STORAGE_KEY);
        if (!raw) return;
        let entries = JSON.parse(raw) as Array<
          [string, {
            pubkey: string;
            dTag?: string;
            aggregateHash?: string;
            capabilities: Capability[];
            blocked: boolean;
            stateQuota?: number;
          }]
        >;

        // Check for old 3-segment keys and migrate if needed
        const hasOldKeys = entries.some(([key]) => key.split(':').length === 3);
        if (hasOldKeys) {
          // Build temporary AclState for migration
          const tempState: import('@kehto/acl').AclState = {
            defaultPolicy: 'permissive',
            entries: Object.fromEntries(
              entries.map(([key, val]) => [key, {
                caps: capArrayToBitfield(val.capabilities),
                blocked: val.blocked,
                quota: val.stateQuota ?? DEFAULT_STATE_QUOTA,
              }])
            ),
          };
          const migrated = migrateAclState(tempState);
          if (migrated !== tempState) {
            // Rebuild entries from migrated state
            entries = Object.entries(migrated.entries).map(([key, entry]) => {
              const parts = key.split(':');
              return [key, {
                pubkey: '',
                dTag: parts[0] ?? '',
                aggregateHash: parts[1] ?? '',
                capabilities: bitfieldToCapArray(entry.caps),
                blocked: entry.blocked,
                stateQuota: entry.quota,
              }] as [string, typeof entries[0][1]];
            });
            // Re-persist migrated data immediately
            localStorage.setItem(STORAGE_KEY, JSON.stringify(entries));
          }
        }

        store.clear();
        for (const [key, val] of entries) {
          if (val.dTag === undefined || val.aggregateHash === undefined) continue;
          store.set(key, {
            key,
            pubkey: val.pubkey,
            dTag: val.dTag,
            aggregateHash: val.aggregateHash,
            capabilities: new Set(val.capabilities),
            blocked: val.blocked,
            stateQuota: val.stateQuota ?? DEFAULT_STATE_QUOTA,
          });
        }
      } catch {
        store.clear();
      }
    }
    ```

    3. Add two helper functions for bitfield conversion (place above the `aclStore` const):
    ```typescript
    import { ALL_CAPABILITIES } from '@napplet/core';
    // Already imported. Add these helpers:

    // Capability name to bit position mapping for bitfield conversion
    const CAP_BITS: Record<string, number> = {
      'relay:read': 1, 'relay:write': 2, 'cache:read': 4, 'cache:write': 8,
      'hotkey:forward': 16, 'sign:event': 32, 'sign:nip04': 64, 'sign:nip44': 128,
      'state:read': 256, 'state:write': 512,
    };

    function capArrayToBitfield(caps: Capability[]): number {
      let bits = 0;
      for (const cap of caps) bits |= (CAP_BITS[cap] ?? 0);
      return bits;
    }

    function bitfieldToCapArray(bits: number): Capability[] {
      return (Object.entries(CAP_BITS)
        .filter(([, bit]) => (bits & bit) !== 0)
        .map(([name]) => name)) as Capability[];
    }
    ```

    Note: Import the `CAP_*` constants from `@kehto/acl` instead of hardcoding if preferred, but the hardcoded map is simpler and avoids importing individual bit constants. The bit values match @kehto/acl/types.ts exactly.
