<objective>
Implement `relay.publishEncrypted` as a napplet→shell path that:

1. Extracts the unsigned event template + recipient pubkey + encryption scheme from the envelope (per @napplet/nub-relay's `RelayPublishEncryptedMessage` shape).
2. Encrypts the plaintext `event.content` using the shell's signer (`hooks.auth.getSigner().nip44.encrypt(recipient, plaintext)` or `.nip04.encrypt(...)`) — SHELL-INTERNAL, napplet never sees these primitives.
3. Signs the encrypted event via `signer.signEvent`.
4. Publishes via the relay pool (same path as relay.publish).
5. Emits `relay.publishEncrypted.result { id, ok, event?, eventId?, error? }`.

Failure modes → `relay.publishEncrypted.result` with `ok: false, error: <message>`:
- Missing signer → `'no signer configured'`
- Missing `recipient` → `'missing recipient'`
- Unsupported `encryption` value → `'unsupported encryption scheme: <value>'` (accept 'nip04' / 'nip44' / omitted=nip44)
- Encryption primitive throws → `'<err.message>'`

**Scope boundaries:**
- This plan is SHELL-MEDIATED ENCRYPTION — napplets have no code path exposing raw plaintext. The former `signer.nip04.*` / `signer.nip44.*` napplet-visible surface was deleted in Plan 12-03 (DRIFT-RT-07). This plan reinstates encryption as SHELL-INTERNAL only.
- ACL gating split (publish vs publishEncrypted with composite `sign:nip44` requirement) lives in Plan 12-10 (DRIFT-ACL-06).

Purpose: Close NUB-08, DRIFT-RT-08, DRIFT-SVC-08, and the SH-C03 "shell-mediates encryption via relay.publishEncrypted" half.

    Insert `case 'publishEncrypted':` between existing `publish` and `query` cases. Add an internal async helper `publishEncryptedHelper`:

    ```ts
    case 'publishEncrypted': {
      const id = m.id ?? '';
      const eventTemplate = m.event; // EventTemplate — unsigned, plaintext content
      const recipient = (msg as NappletMessage & { recipient?: string }).recipient ?? '';
      const encryption = ((msg as NappletMessage & { encryption?: string }).encryption ?? 'nip44') as 'nip04' | 'nip44' | string;

      function reply(ok: boolean, extra: Record<string, unknown> = {}): void {
        hooks.sendToNapplet(windowId, { type: 'relay.publishEncrypted.result', id, ok, ...extra } as NappletMessage);
      }

      if (!recipient) { reply(false, { error: 'missing recipient' }); return; }
      if (encryption !== 'nip44' && encryption !== 'nip04') {
        reply(false, { error: `unsupported encryption scheme: ${encryption}` });
        return;
      }
      const signer = hooks.auth.getSigner();
      if (!signer) { reply(false, { error: 'no signer configured' }); return; }
      if (!eventTemplate || typeof eventTemplate !== 'object') {
        reply(false, { error: 'invalid event template' }); return;
      }

      (async (): Promise<void> => {
        try {
          const plaintext = String((eventTemplate as { content?: unknown }).content ?? '');
          const ciphertext = encryption === 'nip44'
            ? await signer.nip44?.encrypt(recipient, plaintext) ?? ''
            : await signer.nip04?.encrypt(recipient, plaintext) ?? '';
          const eventWithCiphertext = { ...(eventTemplate as object), content: ciphertext } as NostrEvent;
          const signedEvent = await signer.signEvent?.(eventWithCiphertext) ?? null;
          if (!signedEvent) { reply(false, { error: 'signEvent returned null' }); return; }

          // Publish via relay service if registered, else fallback to hooks.relayPool.
          const relayService = serviceRegistry['relay'] ?? serviceRegistry['relay-pool'];
          if (relayService) {
            // Delegate by reusing the existing publish message type; the service
            // sees a synthesized relay.publish envelope with the signed encrypted event.
            const publishMsg = { type: 'relay.publish', id, event: signedEvent } as NappletMessage;
            relayService.handleMessage(windowId, publishMsg, (resp: NappletMessage) => {
              // Translate the service's relay.publish.result back into relay.publishEncrypted.result.
              const r = resp as NappletMessage & { ok?: boolean; accepted?: boolean; eventId?: string; message?: string; error?: string };
              if (r.type === 'relay.publish.result' || r.type?.startsWith('relay.publish')) {
                const okVal = r.ok ?? r.accepted ?? false;
                reply(okVal, {
                  event: signedEvent, eventId: signedEvent.id,
                  ...(okVal ? {} : { error: r.error ?? r.message ?? 'publish failed' }),
                });
              }
            });
          } else if (hooks.relayPool?.isAvailable()) {
            hooks.relayPool.publish(signedEvent);
            reply(true, { event: signedEvent, eventId: signedEvent.id });
          } else {
            reply(false, { error: 'no relay pool available' });
          }

          // Also buffer + deliver to local subscribers (ciphertext, never plaintext).
          eventBuffer.bufferAndDeliver(signedEvent, windowId);
        } catch (err) {
          reply(false, { error: (err as Error).message ?? 'encryption failed' });
        }
      })();

      break;
    }
    ```
