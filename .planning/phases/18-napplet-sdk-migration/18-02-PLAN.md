From @napplet/sdk:

```typescript
import '@napplet/shim';
import { relay, ipc, storage, type EventTemplate } from '@napplet/sdk';

// IPC namespace (chat→bot envelope round-trip):
ipc.emit(topic: string, extraTags?: string[][], content?: string): void
ipc.on(topic: string, callback: (payload: unknown, event: NostrEvent) => void): Subscription

// Storage namespace (chat history persistence):
storage.getItem(key: string): Promise<string | null>
storage.setItem(key: string, value: string): Promise<void>

// Relay namespace (optional showcase per D-03 — NOT load-bearing for NAP-02):
relay.publish(template: EventTemplate, options?: { relay?: boolean }): Promise<NostrEvent>
```

**Key shape note** (verified in shim/src/index.ts):
- `ipc.emit` content arg is a JSON-encoded string; chat already uses `JSON.stringify({ text, timestamp })`.
- `ipc.on` callback receives `payload` already JSON-parsed by the shim — no manual JSON.parse needed.
- The shim auto-handles AUTH; do not write a manual handshake.
