# Phase 84 — Plan 84-01: NIP-5A/5D Resolution Module (`@kehto/nip`)

**Milestone:** v1.20 NIP-5D Content-Addressed Runtime Resolution
**PR:** PR1 (branch `feat/nip5d-resolver`, off `main`) — self-contained, unblocks Phase 85.
**Requirements:** RESOLVE-01..07

## Goal

Add a pure, environment-agnostic NIP-5A/5D resolution surface to `@kehto/nip`:
two new subpaths `@kehto/nip/5a` (aggregate hash) and `@kehto/nip/5d`
(kinds + manifest parse + signature verify + Blossom blob fetch/verify +
pure `resolveNapplet` orchestrator). No runtime/shell/playground changes.

## Approach (TDD — test first, then implement, per task)

### Task 1 — `@kehto/nip/5a` aggregate hash
- `src/5a/index.ts`: `computeAggregateHash(pathTags)` — per NIP-5A: line
  `"<sha256> <abs-path>\n"`, sort ascending, concat UTF-8, sha256 → lowercase hex
  (via `@noble/hashes/sha2.js`). `aggregateTagFromEvent(event)` reads
  `["x","<hex>","aggregate"]`. `verifyAggregate(event)` recomputes from `path`
  tags and compares to the `x` aggregate tag.
- `src/5a/index.test.ts`: pin the NIP-5A example vector
  (`/index.html` `186ea5…1c99`, `/favicon.ico` `fedcba…4321` →
  `c2ff582b672a4c689c5e1753528f03dd31b95ec1fdcc3d82d25e7d91e8769638`); sort
  independence; verifyAggregate accept + reject-on-mismatch.

### Task 2 — `@kehto/nip/5d` kinds + manifest parse
- `src/5d/index.ts`: constants `NAPPLET_KIND_NAMED = 35129`,
  `NAPPLET_KIND_ROOT = 15129`, `NAPPLET_KIND_SNAPSHOT = 5129`,
  `NAPPLET_KINDS` set. `parseNappletManifest(event): NappletManifest`
  → `{ kind, pubkey, dTag, paths: {path,sha256}[], aggregateHash, servers,
  requires, title?, description? }`. Reject malformed (no path tags, bad kind).
- tests: parse a well-formed manifest; reject wrong kind / missing path tags.

### Task 3 — signature verify (RESOLVE-05)
- `verifyManifestSignature(event)` via `nostr-tools/pure` `verifyEvent`.
- tests: validly-signed manifest verifies; tampered sig/content rejected.

### Task 4 — Blossom blob fetch + verify (RESOLVE-06)
- `verifyBlobHash(bytes, sha256)` (sha256 of bytes === expected).
- `fetchBlob(servers, sha256, { fetch })` — try each `server` `GET <server>/<sha256>`,
  verify hash, return `Uint8Array`; reject if none match.
- tests: matching blob accepted; forged blob rejected; all-servers-fail rejected.

### Task 5 — pure `resolveNapplet` orchestrator (RESOLVE-02/04/05/06)
- `resolveNapplet({ event, fetchBlob })` — verify sig → parse → verify aggregate
  → fetch+verify every blob → return `{ dTag, aggregateHash, files: Map<path,bytes>,
  indexHtml }`. Reject (throw typed error) on ANY failure path.
- `assembleIndexHtml(files)` helper (pure) — return the verified `/index.html` text.
- tests: happy path; reject on each of bad-sig / aggregate-mismatch / blob-mismatch /
  missing-index.

### Task 6 — packaging + barrel + docs (RESOLVE-01/07)
- `package.json`: add `./5a` + `./5d` exports, `@noble/hashes ^2.0.0` dep, keywords.
- `tsup.config.ts`: add `src/5a/index.ts`, `src/5d/index.ts` entries.
- `src/index.ts` barrel: re-export `./5a` + `./5d`.
- `vitest.config.ts`: add `@kehto/nip/5a` + `@kehto/nip/5d` aliases.
- `src/5a/README.md`, `src/5d/README.md`; CHANGELOG entry; changeset.
