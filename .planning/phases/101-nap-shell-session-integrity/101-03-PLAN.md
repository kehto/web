| Boundary | Description |
|----------|-------------|
| parent Window → injected shell receiver | Only the hosting parent may establish the frame's local environment. |
| cached environment → untrusted napplet code | Domains/services are readable but must not be mutable or reassignable. |
| string input → support decision | Untrusted runtime values must not broaden support through coercion or approximate matching. |
