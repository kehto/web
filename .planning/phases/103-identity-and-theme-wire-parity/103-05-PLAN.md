## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Paja theme control → retained ThemeService | A host UI choice changes napplet-visible shared theme state. |
| ThemeService callback → ShellBridge | A synchronous service callback crosses into authenticated browser delivery after an adapter-before-bridge construction cycle. |
| ShellBridge → sandboxed target | Complete theme state crosses into an opaque-origin napplet only after session, domain, and current capability eligibility checks. |

| T-103-05-02 | Tampering | ThemeService/host state ordering | medium | mitigate | Keep ThemeService.publishTheme as the sole mutation and assert callback-time getCurrentTheme plus browser-time theme.get equal the changed payload. |
