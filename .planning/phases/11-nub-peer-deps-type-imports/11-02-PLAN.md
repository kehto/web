export interface NubMessage {

The minimum is deliberate. The NUB-02 acceptance test is "no hand-copied duplicate of a nub-specific type," and `NubMessage` is a generic supertype, not a duplicate.

- No new handler logic. No message-shape fixes (those are Phase 12 drift work).
- No refactoring of the dispatch switch (Phase 14).
- No new message types handled (Phase 12 / Phase 13).
