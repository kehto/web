<objective>
Map the `inc` dispatch domain to the SAME capability resolution as `ifc` in the
ACL resolver (D5 / ALIGN-06), so `inc.emit` enforces the `relay:write` gate
instead of falling through the `unknown → null/null` branch and bypassing ACL.

Purpose: A `>=0.9.0` napplet sends `inc.*` wire messages. Without this, the new
dispatch key (Plan 02) would route through the IFC handler but skip the
capability check that `ifc.*` enforces — a security/correctness regression.
Output: `inc` aliased to `ifc` in `resolveCapabilitiesNub`, updated JSDoc table,
and `inc.*` parallel tests alongside the existing `ifc.*` tests.
</objective>
