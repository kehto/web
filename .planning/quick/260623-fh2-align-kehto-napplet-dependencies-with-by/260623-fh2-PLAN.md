# Align Kehto Napplet Dependencies With bytesMany Release

## Goal

Downstream consumers report `resource.bytesMany` is unavailable even though the
Kehto source has host-side handling. Align Kehto's real package manifests with
the published Napplet versions that contain `bytesMany`.

## Scope

- Update Kehto package and fixture `package.json` files from pre-`bytesMany`
  Napplet versions to the current published versions.
- Refresh `pnpm-lock.yaml`.
- Add changesets for Kehto packages whose published dependency surface changes.
- Verify installed package surfaces expose `bytesMany`.

## Non-goals

- Do not edit archived `.claude/worktrees/*` snapshots.
- Do not change wire behavior or invent protocol surface.
