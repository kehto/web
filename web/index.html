An early runtime for sandboxed Nostr napplets

Kehto Runtime

Kehto is a runtime implementation for NIP-5D napplets, not the runtime for the ecosystem. It provides shell-side packages for protocol routing, ACL enforcement, service handlers, gateway loading, and browser integration.

Alpha notice: NIP-5D is still under development and NUB contracts are not final. APIs, capability names, requires declarations, and supports() behavior may change.
