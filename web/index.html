Shell-side runtime for sandboxed napplets

Kehto Runtime

Kehto is a modular runtime implementation for NIP-5D napplets. It gives host applications the shell-side pieces for protocol routing, ACL enforcement, service handlers, gateway loading, and browser integration.

01 Cradle the app

Sandboxed iframes stay contained while the shell owns capabilities.

02 Route the protocol

Runtime packages handle NIP-5D messages, services, and enforcement.

03 Prove the surface

Gateway-hosted napplets and docs show the integration contract.

Alpha notice: NIP-5D is still under development and NUB contracts are not final. APIs, capability names, requires declarations, and supports() behavior may change.
