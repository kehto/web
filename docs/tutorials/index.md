# Tutorials

Tutorials are complete learning paths. Use how-tos when you already know the concept and need one focused task.

| Tutorial | Use when |
|----------|----------|
| [Minimal host shell](./minimal-host-shell.md) | You want the shortest path from install to hosting one napplet. |
| [Runtime implementation](./runtime-implementation.md) | You need to wire adapters, services, ACL, shell bridge, gateway loading, and teardown. |
| [Napplet integration](./napplet-integration.md) | You are writing a napplet that declares `requires`, checks hosted `supports()`, and uses NAP helpers safely. |

Cache implementation is a focused host task rather than a full tutorial. Use
[Implement a napplet artifact cache](../how-tos/implement-napplet-artifact-cache.md)
after the host shell can already resolve and load NIP-5D artifacts.
