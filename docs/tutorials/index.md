# Tutorials

Tutorials are complete learning paths. Use how-tos when you already know the concept and need one focused task.

| Tutorial | Use when |
|----------|----------|
| [Minimal host shell](./minimal-host-shell.md) | You want the shortest path from install to hosting one napplet. |
| [Runtime implementation](./runtime-implementation.md) | You need to wire adapters, services, ACL, shell bridge, gateway loading, and teardown. |
| [Napplet integration](./napplet-integration.md) | You are writing a napplet that declares `requires`, checks hosted `supports()`, and uses NUB helpers safely. |
